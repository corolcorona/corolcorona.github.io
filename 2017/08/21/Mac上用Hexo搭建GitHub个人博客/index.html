mac系统搭建各种开发环境

1.通过nom安装hexo(之前Node.js和Git都安装过)
npm install hexo-cli -g
–/usr/local/bin/hexo

2.cd到一个指定的路径，执行
–cd /Users/corolcorona/Documents/blog
hexo init blog
进行初始化，blog是你建立的文件夹名称

3.cd blog
执行
hexo s
可以进入网址http://localhost:4000查看

4.进入Github，新建仓库，名为：用户名.github.io

5.终端cd到blog文件夹下，vim打开_config.yml，修改最后
deploy:
type: git
repository: https://github.com/用户名/用户名.github.io
branch: master

6.在blog文件夹先执行：npm install hexo-deployer-git —save
再执行部署命令：
hexo generate 或者：hexo g
hexo deploy 或者：hexo d
打开用户名.github.io，查看

7.每次新增文章或者更换主题后，都需要执行
hexo clean //清除缓存文件 (db.json) 和已生成的静态文件 (public)
hexo g //生成缓存和静态文件
hexo d //重新部署到服务器
