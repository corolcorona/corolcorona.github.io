Mac基于Hexo搭建GitHub个人博客

Hexo是一个基于Node.js的静态博客框架,只包含HTML, CSS 和 Javascrip；无数据库，无动态数据，无需通入过多精力维护；

一.安装过程（最新安装过程以官网为准https://hexo.io）
1.通过npm安装hexo(之前Node.js和Git都安装过)
npm install hexo-cli -g

2.cd到一个指定的路径，执行
hexo init blog
进行初始化，blog是你建立的项目名称，会生成以下文件
source：博客资源文件夹
source/_drafts：草稿文件夹
source/_posts：文章文件夹
themes：存放主题的文件夹
themes/landscape：默认的主题
_config.yml：配置文件

3.cd到blog文件夹下
执行
hexo s
可以进入网址http://localhost:4000查看

二.进入Github，新建仓库，名为：用户名.github.io

三.配置hexo
1.cd到blog文件夹下，vim打开_config.yml，修改最后
deploy:
type: git
repository: https://github.com/用户名/用户名.github.io
branch: master

2.在blog文件夹先执行：npm install hexo-deployer-git —save
再执行部署命令：
hexo generate 或者：hexo g
hexo deploy 或者：hexo d
打开用户名.github.io，查看

四.新建博客
hexo new 博客名 //新建博客
hexo clean //清除缓存
hexo g //生成缓存和静态文件
hexo d //重新部署到服务器

五.修改主题
如果要修改主题，只需要将主题文件clone到themes目录下，并修改_config.yml配置文件即可
可以去主题官网https://hexo.io/themes/，每个主题都有相关的GitHub和操作文档。

六.添加分类
1.hexo new page categories

2.根据执行后提示的路径，找到index.md这个文件，修改如下

title: 文章分类
date: 2017-05-27 13:47:40

type: “categories”

3.给文章添加“categories”属性

title: Mac上用Hexo搭建GitHub个人博客
date: 2017-05-26 12:12:57
categories:
